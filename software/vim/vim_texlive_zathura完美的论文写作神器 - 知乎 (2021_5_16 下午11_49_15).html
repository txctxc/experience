vim+texlive+zathura完美的论文写作神器

dremofly

Latex是论文写作的神器。Vim在文本编辑器领域也一直处于独树一帜，难以撼动的地位。这两个的结合之前在网络上爆红了一把世界上最好的编辑器Vim：1700多页数学笔记是如何实时完成的_LaTex。它们两个的结合还是颇具吸引力的，因此我也想把它们移植到我的机子上。

之前一直使用的是vscode作为Latex的编辑器，vscode上安装了vim语法插件，书写体验也是非常的顺畅。但是本着一颗折腾的心，所以想把电脑上的vim也安装latex插件。

系统版本为ubuntu18.04。

安装texlive

texlive是由国际TEX用户组（TEXUsers Group，TUG）整理和发布的TEX软件发行套装，包含与TEX系统相关的各种程序、编辑与查看工具、常用宏包及文档、常用字体及多国语言支持。使用这个环境可以

安装方式为在命令行中输入：

sudo apt-get install texlive-full

安装的时间可能会比较长。

在vim中安装插件

在~/.vimrc(vim的配置文件)中添加以下代码，

"{{ vimtex
Plug 'lervag/vimtex'

let g:tex_flavor = 'latex'

" use vimtex as default compiler
let g:vimtex_compiler_latexmk_engines={'_':'-xelatex'}
let g:vimtex_compiler_latexrun_engines={'_':'xelatex'}

" use zathura as the default pdf reviewer
let g:vimtex_view_method='zathura'

" show the compiler hint
let g:vimtex_quickfix_mode = 1

" hide the last two lines
set conceallevel=1

let g:tex_conceal='abdmg'
"}}

之后在vim中运行，一下命令以安装插件。如果在安装的过程中没有看到vimtex的话，就退出vim，重新进入，然后运行下面命令

:PlugInstall

其中这里编译文件使用的是xelatex，因为其支持中文。还可以将其替换成其他的插件。

另外，预览所使用的pdf阅读器为zathura，它是一款支持类vim快捷键的编辑器，和vim搭配起来可以在编辑latex的时候实现纯粹的键盘操作。 zathura也可以进行一些配置，这里就不赘述了。 zathura的安装命令为：

sudo apt install xdotool zathura

运行

接下来就可以进行使用了。使用vim打开一个tex文档，在normal模式下输入

\ll

可以使得文档的编译进入continuous模式，也就是说当tex文档被保存时，会自动对其进行编译并输出为pdf。

\lc为清除编译文件，应该也会比较常用到。
